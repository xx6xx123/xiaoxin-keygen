#  Xiaoxin Codex Keygen

> 一款用于注册 Codex 免费账户的工具，内置预设配置方案（线路）和 CPA 信息。

---

## 📌 简介

**Xiaoxin Codex** 是一款轻量级注册工具，专为快速注册 **Codex 免费账户** 设计。  
内置多套配置方案（线路），自动加载 CPA 信息，一键开始注册。

---

## ⚡ 快速开始

### 1. 选择配置方案
- 打开工具
- 在界面选择适合的 **配置方案（线路）**
- 系统自动加载预设 CPA 信息

### 2. 开始注册
- 点击“开始注册”按钮
- 等待完成即可获得 Codex 免费账户并传至cpa

### 3. 自定义 CPA 设置（可选）

> 如需使用自己的 CPA 地址：

1. 点击右上角 **⚙️ 齿轮图标**
2. 输入有效的 **激活码**
3. 点击确认，切换为自定义 CPA 配置

### 4.插件安装方法
下载 xiaoxin-keygen.zip​ 文件
打开浏览器，进入扩展程序管理页面：
chrome://extensions/   (Chrome)
edge://extensions/     (Edge)
开启 开发者模式（右上角开关）
直接将 xiaoxin-keygen.zip​ 文件拖入页面
点击确认安装，即可完成
